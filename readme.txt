It adds a widget that displays random Einstein quotes every time you refresh the page. Your theme must support widgets.


== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
the top-level folder.
1. Activate the plugin through the 'Plugins' menu in WordPress
