It adds a widget that displays random Einstein quotes every time you refresh the page. Your theme must support widgets.


== Installation ==

1. Upload `EinsteinQuotes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
